Museum Cases
Installed in the Specialty Filaments Building during the 2006 S.E.A.B.A. Art Hop, Burlington, VT, September 8 - 22, 2006

The configuration of the eight Museum Cases is flexible, so each space in which they are installed becomes a new “museum.” In this installation at the Art Hop, six cases have been set up in a formal grid that still allows visitors to circulate around the cases to view them from all sides.
